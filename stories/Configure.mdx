# StandardAI CC Design System

コールセンター業務改善AI「オペすいすい」のためのデザインシステムです。
Ant Design をベースに、プロジェクト固有のデザイントークンとコンポーネントパターンを定義しています。

## 技術スタック

<ul>
  <li><strong>UI フレームワーク</strong>: Ant Design v6</li>
  <li><strong>スタイリング</strong>: Tailwind CSS + CSS カスタムプロパティ</li>
  <li><strong>デザイントークン</strong>: Tokens Studio (DTCG format) → Style Dictionary</li>
  <li><strong>Figma 連携</strong>: Tokens Studio プラグインで双方向同期</li>
  <li><strong>テスト</strong>: Vitest + Playwright (Storybook addon-vitest)</li>
</ul>

## デザイントークン

トークンは3層構造で管理されています。全て `--sai` プレフィックスの CSS カスタムプロパティとして出力されます。

<table className="sb-table">
  <thead>
    <tr><th>レイヤー</th><th>ファイル</th><th>説明</th></tr>
  </thead>
  <tbody>
    <tr><td><strong>Global</strong></td><td><code>{'tokens/global/*.json'}</code></td><td>生の値（カラースケール、フォントサイズ、スペーシング等）</td></tr>
    <tr><td><strong>Semantic</strong></td><td><code>{'tokens/semantic/*.json'}</code></td><td>用途別エイリアス（text.primary → neutral.800）</td></tr>
    <tr><td><strong>Component</strong></td><td><code>{'tokens/component/*.json'}</code></td><td>コンポーネント固有の上書き（必要に応じて追加）</td></tr>
  </tbody>
</table>

トークン変更後のビルド:

```bash
npm run build:tokens
```

出力先:
- `src/tokens/variables.css` — CSS カスタムプロパティ
- `src/tokens/tokens.ts` — TypeScript 定数

## Ant Design テーマ

Ant Design は `ConfigProvider` 経由でデザイントークンが適用されています。
テーマ定義は `src/theme/antd-theme.ts` にあります。

```tsx
import { ConfigProvider } from 'antd';
import { antdTheme } from '@standardai/design-system/theme';

<ConfigProvider theme={antdTheme}>
  <App />
</ConfigProvider>
```

## 開発ガイド

### セットアップ

```bash
npm install
npm run build:tokens
docker compose up -d    # Storybook 起動
# → http://localhost:6006
```

### ディレクトリ構成

```
tokens/         デザイントークン (DTCG JSON)
src/components/ 独自コンポーネント
src/theme/      Ant Design テーマ設定
src/tokens/     生成物 (自動生成、編集不可)
stories/        Storybook ストーリー + ドキュメント
```

### コンポーネント追加手順

1. `src/components/ComponentName/` にコンポーネントを作成
2. `stories/components/ComponentName.stories.tsx` にストーリーを追加
3. デザイントークンの CSS 変数 (`var(--sai…)`) を使用してスタイリング

### Figma 連携

Figma の Tokens Studio プラグインで `tokens/` ディレクトリと同期します。
トークン変更後は `npm run build:tokens` で CSS/TS に反映してください。
